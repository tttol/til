# 半導体の歴史
- 90's 半導体はインテルが覇権を握っていた
- 2007年iPhoneが誕生
- インテルはAppleと揉めて、iPhoneのCPUの半導体チップを作る契約はしなかった
- ムーアの法則
  - 半導体の集積率は18ヶ月で2倍になる
  - つまり、コストは半分になる
  - 半導体の集積率は指数関数的に伸びていく
- 半導体は設計と製造で分業化される
  - 設計
    - NVIDIA
    - ARM
  - 製造
    - TSMC
- インテルは分業化せず、設計→製造を一気通貫でやる
  - となると、インテルは自社のデバイスのチップしか作れなくなる
  - iPhoneのチップを作れない
- ARMはAppleのチップの設計を担当することになる

# CPU
- Macは2000年初頭PowerPC CPUというCPUを使っていた
- PowerPC CPUは薄型のMacbookに搭載するには消費電力と発熱が大きすぎた
- 2005年Intel製CPUに乗り換えて、macbookを無事発表
- しかしIntel製CPUは供給制約やリリースの遅れがあり、Appleのロードマップにも影響が出る
- AppleはARMと手を組んで2008年頃から自社CPUの開発に取り組みだす
- 2020年にM1チップを発表
